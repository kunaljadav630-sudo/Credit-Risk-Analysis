# 💳 Credit Risk Analysis Dashboard

## 📌 Project Overview
Developed a Credit Risk Analysis Dashboard using Excel, SQL, and Power BI to analyze customer credit behavior, loan performance, and default risk patterns. The project focuses on identifying high-risk customers, understanding loan trends, and supporting data-driven financial decisions.

## 🛠️ Tools Used
- Excel
- SQL
- Power BI
- DAX
- Power Query

## 📊 Key KPIs
- Total Customers
- Total Loans
- Default Rate
- Average Loan Amount
- Total Funded Amount
- Total Recovered Amount
- Risk Distribution

## 📈 Dashboard Features
- KPI Cards
- Bar Charts
- Line Charts
- Matrix Tables
- Slicers
- Interactive Filters
- Customer Segmentation
- Loan Status Analysis

## 📂 Files Included
- Credit Risk Dataset (.csv)
- SQL Queries (.sql)
- Power BI Dashboard (.pbix)
- Dashboard Screenshots (.png)

## 📷 Dashboard Preview
Dashboard screenshots are available in this repository.

## 🎯 Project Objective
To analyze customer credit profiles, evaluate loan performance, and identify potential default risks through interactive dashboards and data visualization.

Data Cleaning | SQL | Power BI | DAX | Power Query | Data Visualization | Dashboard Development | Credit Risk Analysis
